# pbmuteurt plugin for UrT
# Plugin for B3 (www.bigbrotherbot.net)

# www.ptitbigorneau.fr

pbmuteurt plugin b3 (v1.3.1) for Urban Terror

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation
------------


1. Copy the 'pbmuteurt' folder into 'b3/extplugins' and 'pbmuteurt.ini' file into '/b3/extplugins/conf'.

2. Open your B3.ini or b3.xml file (default in b3/conf) and add the next line in the [plugins] section of the file:
    for b3.xml
        <plugin name="pbmuteurt" config="@b3/extplugins/conf/pbmuteurt.ini"/>
    for b3.ini
        pbmuteurt: @b3/extplugins/conf/pbmuteurt.ini
 

3. create the pbmuteurt table in your database importing the pbmuteurt.sql file.
